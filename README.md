### What is git?
* version control to track file chnages and save versions of files.
* Git commands:
    * CD, ls, rm dir. 
    * git init, git add . , git commit -m"message" 
* States of files on git :
    * Modified:(untracked,red)
    * Staged: (tracked, added to index, green)
    * committed
